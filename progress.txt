**Brand decisions locked:**
- One-liner: **"Titrate — Sovereign airdrop tooling. Sign cold, run local, deploy anywhere."** Replaces the inaccurate "offline-first" framing.
- Logo: custom SVG titration curve — angular `_/` path (flat low plateau → diagonal rise → flat high plateau) with a hollow equivalence-point circle at the inflection. `viewBox` 170×150, stroke-width 12 path, hollow circle r=22 stroke-8. Large hollow chosen because it survives shrinking to 22px nav scale. TUI falls back to `∫` (U+222B); terminal splash uses the three-line ASCII `_/●` banner.
- Palette: ink (ops surfaces), cream (brutalist surfaces), `pink-400/500/600/700` (role-split: 500 non-text/mark, 600 text-on-fill + CTA, 400 on-dark, 700 hover). All text pairings pass WCAG AA 1.4.3 + 1.4.11.
- Typography: IBM Plex Sans + Plex Mono + Plex Serif (Google Fonts).
- Mode split: TUI = operator-A always, dark-only. Web GUI = brutalist-B for decisions (landing, setup, summaries, settings), operator-A for observation (live pipeline, batch tables, logs, diagnostics) — embedded `OperatorPanel` wraps the latter with a 2px + 4px offset seam inside brutalist chrome.
- Dark-mode marketing in scope as the "Dark brutalist" surface variant, honors `prefers-color-scheme: dark` + manual toggle via `data-theme` attribute.
- Motion: 3 durations (80/150/240ms) × 3 easings × 6-primitive catalog + reduced-motion rule that cuts decorative motion while preserving focus, fades, and color transitions.
